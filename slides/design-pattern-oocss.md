##  OOCSS

- Separate structure and skin
```
.box {
    width: 100px;
    margin: 10px 20px;
}
.box-shadow {
    color: #000;
    border: 1px solid #f0f;
}
```
- Separate container and content
```
.container {
    width: 1000px;
    overflow: hidden;
}
.content {
    text-align: center;
    padding: 10px;
}
```
